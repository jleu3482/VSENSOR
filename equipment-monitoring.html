Install the sensor

The purpose is to find the average vibration and/or the peak vibration of a mechanical component so that the engineer can plot the vibration trend through time.

1. Install the sensor on the mechanical component. It is highly recommended to use Loctite 495 or any cyanoacrylate based adhesive at the base of the sensor to stick on your train component surface, however can also use strap ties or both methods of attachment.

2. When the sensor is stationary, press the user button once to zero-out the gravity offset and the data collection will start immediately.  

3. Now start the machine / equipment to measure its vibration.
